# Avances 28/03/2026 — Frontend Dashboard NX-8

## Resumen
Se ha creado un dashboard web completo para visualizar y controlar el simulador de central de intrusión NX-8 en tiempo real.

## Detalles Técnicos

### Stack Frontend
- HTML5 semántico
- CSS vanilla con tema oscuro SCADA (~650 líneas)
- JavaScript vanilla con polling API (~350 líneas)
- Fuentes: Inter + JetBrains Mono (Google Fonts)

### Integración con Backend
- Se registró `@fastify/static` para servir archivos desde `public/`
- Se registró `@fastify/cors` para desarrollo
- El dashboard se accede en `http://localhost:8080` (mismo puerto que la API)
- Polling automático: estado del panel cada 2s, eventos cada 3s, escenarios cada 10s

### Archivos Creados
- `public/index.html` — Estructura HTML del dashboard
- `public/css/style.css` — Estilos completos con glassmorphism y animaciones
- `public/js/app.js` — Lógica de la aplicación

### Archivos Modificados
- `src/control-api/server.ts` — Registro de plugins estáticos y CORS

### Funcionalidades
1. Monitorización en tiempo real de 6 indicadores del panel
2. Visualización de 16 zonas con filtros y modal de acciones
3. Control de 2 particiones (armar/desarmar)
4. Feed de eventos en vivo
5. Inyección de fallos (AC, batería, tamper, comm)
6. Gestión de escenarios
7. Modo laboratorio (delay, jitter, ráfagas)
8. Toast notifications
9. Diseño responsive
